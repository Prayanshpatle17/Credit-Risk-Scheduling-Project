# Credit-Risk-Scheduling-Project
This project focuses on building a Credit Risk Scheduling Model that helps financial institutions evaluate the risk associated with loan applicants and optimize repayment scheduling.

The model predicts whether a customer is likely to default and helps in designing better loan approval and repayment strategies.
Objectives:-
1. Predict credit risk (Good / Bad customer)
2. Perform feature engineering on financial data
3. Build and evaluate machine learning models
4. Optimize repayment scheduling based on risk
5. Visualize insights for decision making
